--------------------------------
	INSTALLATION

Move all files to C:/Games/Heroes of Might and Magic 3 Complete Edition/
If you want to play with the "Landscapes" plugin, double-click "H3L_HotA_Patcher.exe"
Then just open HexSwapper.exe, and you're all set!

--------------------------------
	OPTIONAL FILES
Optional files:
Data - "streamer request" skins. 
noLandscapes - a version of the h3hota HD.exe to replace the one in your main game folder, should you choose to uninstall the Landscapes plugin. 
safeHDmod - a version of the HD mod which is confirmed to work well with Pumpkin Patch, in case a future HD mod update should cause any issues. 
PP.ini - a file to restore all basic Pumpkin Patch settings. 

--------------------------------
	HEXSWAPPER

The HexSwapper is a new program that you can run to modify your game. 
The optional features include skins, gameplay changes and reverting some features to the original Heroes 3. Simply open the program, *without enteirng the game*, to modify your version of the Pumpkin Patch. 
If you want to play online with friends, you can all have different skins, but MAKE SURE THE GAMEPLAY CHANGES ARE THE SAME!  Otherwise, you may experience various crashes or bugs.
